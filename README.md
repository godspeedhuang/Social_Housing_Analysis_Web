## 專案簡介
「設宅配方」為促使社會住宅公共空間使用效率最大化，將設施需求程度列為評估指標， 如社宅附屬設施之托幼中心、長照中心，及周邊居民需求之青創空間、圖書室等。 社宅規劃單位可依照所蒐集到的社宅申請需求、當地居民需求回饋，換算權重後輸入數值， 系統將計算推薦最適的公共設施空間排列組合，快速輔佐社宅規劃人員進行決策。
